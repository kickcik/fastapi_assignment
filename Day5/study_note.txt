3. :=(Walrus Operator, 바다코끼리 연산자)
- 기본적으로 할당연산자이나, 값을 반환해준다.
- 파이썬의 특성으로 None은 Falsy하기 때문에
    if value := get_somthing():
    ...
    과 같은 방법으로 True, False 처리를 통해 실행문을 작성할 수 있다.
    특히 API가 클라이언트로부터 받은 값에 대한 검증을 시도할 때 아주 요긴하게 쓰인다.
    하지만, 저런 필터링으로는 return type이 None이 아닌 모든 함수들에 대해서, return 누락 문제가 발생하기 때문에
    raise와 같은 에러 발생문을 통해 return 누락을 해결할 수 있다.
